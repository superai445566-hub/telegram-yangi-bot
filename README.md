# Telegram yangi Worker Bot

Bu bot ishchilarni ro'yxatdan o'tkazish uchun yaratilgan.

## Funksiyalar:
- Ishchi ro'yxatdan o'tkazish
- Admin paneli
- Ma'lumotlarni bazada saqlash
- Tasdiqlash tizimi

## Render.com da deploy qilish uchun:
1. Ushbu repository ni Render.com ga ulang
2. Web service yarating
3. Bot avtomatik ishga tushadi
